# SKEA Rails

Welcome to Seth & Ken's Excellent Adventure - Rails edition!!!

This is a sample app used to perform an authz functions exercise

* Ruby version

whatever is in `.ruby-version` :smile:

* System dependencies

Your system should have `rvm` or `rbenv`

* Configuration

Run:

```
# To get the db setup
rake db:setup

# To start the thing
rails s
```
